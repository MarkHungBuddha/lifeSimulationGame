# 蒙地卡羅人生模擬遊戲

一款網頁退休財務模擬遊戲。核心體驗：讓玩家觀察「現在的決策」在「長期不確定性」下的結果。

不使用常態分佈假設，而是以真實歷史資料 **Block Bootstrap 重抽樣**，保留資產間真實的相關性與肥尾分佈。

## 特色

- **歷史資料驅動** — 1972-2023 共 52 年，涵蓋 S&P 500、美國長債、黃金、現金、REITs、CPI
- **Block Bootstrap** — 連續區塊抽樣（block size=4），保留景氣循環序列相關性
- **可重現模擬** — xoshiro128** seeded PRNG，同一 seed 永遠產出相同結果
- **三國模擬** — 美國 (USD)、台灣 (TWD)、日本 (JPY) 各有獨立參數與事件
- **移民模組** — 台灣→日本/美國多階段移民模擬（簽證、定居、永住）
- **購屋模組** — 自住房購買模擬，房貸、持有成本、房價漲跌
- **職業系統** — 10 大職業分類 × 3 國薪資，年度加薪隨年齡遞減，職業專屬事件
- **隨機事件** — 三國各 80+ 種事件（市場、職涯、健康、家庭、財產、法律、職業專屬）
- **雙模式** — 批次蒙地卡羅模擬 + 單條路徑人生故事
- **三種提領策略** — 4% 法則、固定金額、動態提領
- **儲存紀錄** — localStorage 儲存/載入模擬參數組合，可命名、比較、重新套用
- **滑桿直接輸入** — 點擊數值可直接鍵入精確數字
- **Dark Mode** — 支援深色模式
- **響應式** — 桌面與手機版自適應
- **純前端** — 無後端依賴，JSON 靜態打包

## 技術棧

- **語言**：TypeScript
- **前端**：React 19 + MUI v7
- **狀態管理**：Zustand
- **圖表**：Canvas API（Percentile Band Chart）
- **運算**：Web Worker（10,000 路徑平行模擬）
- **RNG**：xoshiro128**
- **建構工具**：Vite 8
- **測試**：Vitest

### 2026-03-24 — v0.7：購屋模組 + 事件系統整合

#### 新增：自住房購買模組
- 三國購屋模擬引擎（`housingEngine.ts`、`housingData.ts`、`housingTypes.ts`）
- 支援自訂房價所得比、頭期款比例、房貸年限
- 購屋當年從投資組合扣除頭期款+交易成本
- 工作期間房貸+持有成本從年投資額扣除，退休後加入提領額
- 房價隨機漲跌（Box-Muller 常態分布）
- UI 控制面板：購屋年齡、房價所得比、頭期款、房貸年限、即時預估
- 人生故事模式顯示購屋結果（房屋市值、淨值、房貸餘額）
